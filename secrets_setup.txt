# GitHub Secrets Setup Guide

1. DOCKER_USERNAME
- Value: [Your Docker Hub username]
- Get this from: https://hub.docker.com/settings/general

2. DOCKER_PASSWORD
- Value: [Your Docker Hub password or access token]
- Create a new access token: https://hub.docker.com/settings/security

3. APP_SECRET_KEY
- Value: 09552264db8f3d74667a0f1f2f658b71b66c85a54d14fbdd26365ff4e84bdf07
- This is a secure randomly generated key

4. GCP_PROJECT_ID
- Value: [Will be created after Google Cloud SDK setup]

5. GCP_SA_KEY
- Value: [Will be created after service account setup]

Steps to add secrets to GitHub:
1. Go to: https://github.com/om-369/defect-detection/settings/secrets/actions
2. Click "New repository secret"
3. Add each secret one by one
